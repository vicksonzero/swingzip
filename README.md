# How to play

Touch screen:

- **Tap-hold** to swing
- **Double tap** to zip to point
- **Tap shortly** to jump / air dash
- **Swipe or use on-screen joystick** to fine-tune

PC:

- **Click-hold** to swing
- **Double-click** to zip to point
- **Click shortly** to jump / air dash
- **WASD / Arrows** to fine-tune
